# E-Commerce-SQL-Analysis

## Overview
This project analyzes an e-commerce dataset using SQL to understand sales trends and customer behavior.

## Dataset
The dataset contains transactional retail data with the following columns:
InvoiceNo, StockCode, Description, Quantity, InvoiceDate, UnitPrice, CustomerID, Country

## Key Questions
- Which countries generate the most revenue?
- Who are the top customers?
- What products sell the most?
- How much customer data is missing?

## Key Insights
- UK generates the majority of sales.
- Some transactions have missing CustomerID values.
- A small number of customers drive most revenue.
